This project is an introduction and demonstration of how powerful and useful CSS is to web design. HTML provides a bare structure for your content. CSS styles the HTML contents based on element names, id and classes. CSS styles can be inserted to the HTML content in three different ways:

  • External CSS
  • Internal CSS
  • Inline

External CSS

External CSS is inserted to a HTML document with the link element inside the head area of an HTML document.

Internal CSS

Internal CSS is inserted to a HTML document with the style element inside the head area of an HTML element.

Inline CSS

Inline CSS is added directly to a HTML element via its style attribute

Cascading Order

CSS styles cascade into one another in the following scheme based on their level of priority:

  1. Browser default
  2. External stylesheet
  3. Internal style
  4. Inline style
